    Outils

    Les lien ci-dessous liste des outils tres utiles a la conception de site internet

  • L'un des outils les plus utile et utiliser par la communauter s'appele Git hub qui est une solution de versionnage de project il est aussi le leader de l'open source en effet n'importe qui peut acceder a des "Repository" ,les modifier ou les reutiliser,ce site a etait realiser avec l'aide de gitHub
  • Pour tester la partie CSS de votre site le site du W3C met a dispostion le Validateur CSS
  • Schnaps.it et un generateur de template qui ne fournit pas un design mais cree un base CSS predefini selon certaine condition
  • mydevice.io est un panneau de controle qui permet de connaitre les resolutions exacte de chaque appareil pour adapter l'affichage en fonction des different appareils
  • Can I Use est une bible incontournable pour connaître toutes les compatibilités navigateurs dans des tableaux détaillés et mis à jour très rapidement
  • Cubic Bezier est un outil très utile pour paramétrer les Transitions CSS. Un graphique permet de déformer la courbe d'accélération à la souris et de générer le code correspondant.
  • MobileHTML est un tableau de compatibilité récapitulatif à destination des plate-formes mobiles.
